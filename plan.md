# 优选商城 开发计划

> 最后更新：2026-07-18

---

## 已完成

### 基础架构
- [x] Spring Boot 2.7 改造（WAR → JAR，内嵌 Tomcat + 前端）
- [x] JWT 鉴权（纯 JDK 实现）
- [x] BCrypt 密码加密 + 明文兼容自动升级

### 用户系统
- [x] 注册 / 登录 / 退出
- [x] 登录状态持久化（localStorage）
- [x] 管理员也可通过商城登录
- [x] 权限区分（普通用户不显示后台入口）

### 商品模块
- [x] 商品列表 + 分页加载（每页 8 件）
- [x] 分类筛选导航
- [x] 模糊搜索（名称 + 品牌）
- [x] 图片展示 + 一键上传更新

### 购物车
- [x] 未登录 localStorage / 已登录数据库双写
- [x] 登录自动合并

### 订单流程
- [x] 确认订单页 + 提交订单
- [x] 我的订单列表 + 状态标签
- [x] 付款 / 取消 / 确认收货

### 个人中心
- [x] 收货地址 CRUD + 默认地址
- [x] 修改密码
- [x] 我的收藏列表

### 互动功能
- [x] 五星打分 + 文字评价 + 匿名评价
- [x] 收藏 / 取消收藏（详情页按钮）

### 后台管理
- [x] 商品管理（列表 + 类型）
- [x] 订单管理（查询 + 创建）
- [x] 用户管理（用户列表）
- [x] 权限菜单树

### UI
- [x] 淘宝橙主色调设计系统
- [x] 全中文界面
- [x] 底部链接全可用
- [x] 浏览器缓存禁用
